# TIC-TAC-TOE   <br>

The popular tic-tac-toe game written in C programming language <br>
This game is played between two players locally <br>
To play this game compile this code and run <br>
You will get the board and positions where you can put the 'X' and 'O' <br>
Players need to enter the respective block numbers where they want to insert their pawns <br>
The first player to get three pawns in a straight (horizontal , staright or diagonal line) wins the game
